Corrections to the sketch

1. SWING2 does not drink from Finnhub. Your diagram fed the SWING2 funnel from FinnhubFeed. Deliberately not so: swing_v2 has its own pipe to Alpaca's data API — partly to cost the Finnhub budget nothing, and partly because Finnhub's Bars carry no open prices, and swing_v2's candle patterns (engulfing, hammer) cannot be computed without opens.
2. "SWING2 (UNP, PLD…)" is a mislabel. UNP, PLD, UNH, UPS are old swing's held positions. swing_v2 holds nothing and can hold nothing — it writes phantom trades to the audit trail and refuses live mode in code.
3. The sketch ends where the hard part begins. "Your trading decisions" is the midpoint, not the output. Below it sit the execution layer (where the July 6/16/17 incidents were fought and fixed) and the memory layer (which exists because Railway wipes logs and disk on every redeploy). A map without them explains the bot's ideas but not its scars.
4. GitHub and Railway were missing. Every behavior switch — the bench, the cap, the regime filter — lives as a Railway env var, which is why most changes need no code push to reverse.

The four pillars and their uses

PillarRoleWhat it's used for here
GitHubBlueprintSource of truth for code and docs; a push is a deploy. Pushed after hours by policy so redeploys land in a dead market.
RailwayBodyRuns the one loop 24/7; holds secrets and switches; mounts /data, the only disk that survives. Known boot personality: cold-cache fetch → brief 429 burst → breakers recover in ~5 min.
FinnhubEyes (paid)All live-strategy data: daily bars, 1-min bars, quotes — behind a rate limiter and per-endpoint circuit breakers. The scale contract (risk numbers from daily bars only) lives at this boundary.
AlpacaHands + second eyesTrading API executes and holds the GTC protective stops 24/7; boot reconcile treats Alpaca as truth. Its free data API separately feeds swing_v2 and the local backtests.

One sentence: GitHub is the blueprint, Railway is the body, Finnhub is the eyes, the engines are the brain, Alpaca is the hands, and /data is the memory — and the sketch's "trading decision" is merely where the brain hands off to the hands.
